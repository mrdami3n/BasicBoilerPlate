# Basic Boiler Plate

This is a basic boilerplate for a Node.js project using TypeScript.

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

Clone the repository
   ```bash
   git clone https://github.com/mrdami3n/BasicBoilerPlate
   cd Basic Boilerplate

## Running the Project

#Install dependencies:

```bash
npm install

#Build the project:

```bash
npm run build

#Start the project:

```bash
npm start

#For development mode:

```bash
npm run dev
